# Capturas de Ecrã

Coloca aqui os ficheiros de imagem da aplicação, com estes nomes exatos (usados no README principal):

| Ficheiro | Ecrã a capturar |
|---|---|
| `feed.png` | Feed Principal (Home) |
| `explorar.png` | Explorar (pesquisa de livros/pessoas) |
| `perfil.png` | Perfil do utilizador |
| `publicacao.png` | Detalhe de uma publicação com comentários |
| `chat.png` | Conversa / lista de mensagens |
| `dark.png` | Aplicação em modo escuro |

**Como tirar as capturas:**
1. Abre a app em [liltymobile.web.app](https://liltymobile.web.app).
2. Em cada ecrã, faz uma captura (no telemóvel ou no navegador).
3. Guarda cada imagem com o nome correspondente da tabela, dentro desta pasta.
4. Faz commit e push:
   ```bash
   git add screenshots/
   git commit -m "Adiciona capturas de ecrã"
   git push
   ```

> Formato recomendado: `.png`, na vertical (formato telemóvel), com largura semelhante entre todas para a galeria ficar alinhada.
